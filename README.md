# Waypoint unity
Project Unity di Matkul Kecerdasan Buatan
Waypoint di Unity adalah titik referensi atau lokasi spesifik (berupa GameObject kosong atau posisi koordinat) yang digunakan untuk mendefinisikan rute, jalur patroli AI, atau tujuan pergerakan objek. Waypoint membantu mengarahkan objek dari satu titik ke titik lain secara berurutan atau otomatis dalam scene game
